🏛️ Civic Tech
🏠
Live · harristaxappeals.org
HarrisTaxAppeals
County-scale property tax protest tool covering 1.1M+ Harris County residential parcels — scaled from a 2,172-parcel Jersey Village prototype. Uses §41.43(b)(3) comparable-home methodology: same HCAD neighborhood code, ±15% sq ft, ±10 years age, geographically closest comps. Enterprise-grade Cloudflare infrastructure: R2 + Workers + PMTiles + gzip compression + rate limiting.
Cloudflare Pages DuckDB PMTiles MapLibre GL JS R2 + Workers Python Pipeline Sharded JSON
